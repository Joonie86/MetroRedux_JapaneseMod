# MetroRedux_JapaneseMod

Metro:Redux 日本語化 mod<br>
■注意<br>
本modはフルHDのみの対応になっています<br>
他の解像度では文字がかなりぼやけてしまうため，フルHDに切り替えてプレイしてください．<br>
<br>
■使い方<br>
以下の2つのファイルをゲームフォルダのルートに上書きコピーしてください．<br>
content.vfx<br>
content99.vfs0<br>

これで日本語化は終了です．<br>
